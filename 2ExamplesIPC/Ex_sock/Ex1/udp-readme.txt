Для реализации аналогичного взаимодействия с использованием протокола UDP 
необходимо внести следующие изменения:
1.	Изменить тип сокета с SOCK_STREAM на SOCK_DGRAM.
2.	Заменить функции recv() и send() на recvfrom() и sendto() соответственно.
Такие изменения позволят программе использовать протокол UDP для обмена данными 
(без установления соединения и без гарантии доставки пакетов в порядке их отправки).
