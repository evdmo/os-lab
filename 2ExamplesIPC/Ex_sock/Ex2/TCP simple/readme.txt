ТЗ: реализовать простейшее клиент-серверное приложение, в котором
при запуске сервер сначала ожидает соединения (запуска клиента), 
после чего он ждёт получения сообщения; 
После запуска клиента, введённое пользователем сообщение отправляется серверу посредством сокета, 
сервер принимает данное сообщение и завершает работу.

    Для передачи информации посредством сокетов в сетевом режиме 
требуется изменить IP-адрес в приложении клиента на адрес сервера в сети. 
Если в качестве сервера будет выбран компьютер, IP-адрес сети можно узнать при помощи утилиты hostname с ключом -i, например, 
	$ hostname -i
	127.0.1.1
или используя иные утилиты.
    В приведенном примере кода программы при определении глобальной константы SERVER_IP следует указать IP-адрес сервера.
